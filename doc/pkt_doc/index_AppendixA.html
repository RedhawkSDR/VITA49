A. Appendix A

This appendix provides definitions of a number of packet classes for use in association with the "VITA 49 Spectrum Survey Profile". All packet classes defined in Appendix A use the "internal use" OUI FF-FF-FA in accordance with VRT-Mod. These packet class definitions may be freely used by any individual or organization, provided that the rules for "internal use" OUIs are satisfied.

Except for the Standard Data Packet, by convention, the Class Identifiers all start with an indication of the year in which they were defined and/or revised. For example, all packet types defined in 2011 will be between FF-FF-FA:2011.0000 and FF-FF-FA:2011.FFFF.
